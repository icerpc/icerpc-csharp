# IceRPC Service Generator

IceRpc.ServiceGenerator provides a [source generator] that implements `IceRpc.IDispatcher` for any partial class
decorated with the `[IceRpc.Service]` attribute.

[Package][package] | [Source code][source]

This package is a dependency of the [IceRPC + Slice][icerpc+slice] and [IceRPC + Ice][icerpc+ice] integrations.
