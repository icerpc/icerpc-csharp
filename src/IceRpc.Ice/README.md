# IceRPC + Ice integration

The IceRPC framework allows you to make RPCs with the serialization format and [IDL] of your choice. It provides full
support for Slice and Protobuf.

TODO: the Ice support is for interop with Ice

The IceRpc.Ice NuGet package is part of the IceRPC + Ice integration, and includes two .NET assemblies:

- IceRpc.Ice.dll is used by the code generated by the [Ice compiler for C#][ice-tools]
- IceRpc.Ice.Generators.dll provides a [source generator]

## Sample Code

```ice
// Ice contract

module VisitorCenter
{
    /// Represents a simple greeter.
    interface Greeter
    {
        /// Creates a personalized greeting.
        /// @param name The name of the person to greet.
        /// @return The greeting.
        string greet(string name);
    }
}
```

```csharp
// Client application

using IceRpc;
using VisitorCenter;

await using var connection = new ClientConnection(new Uri("icerpc://localhost"));

// GreeterProxy is a struct generated by the Ice compiler; its implementation uses
// IceRpc.Ice.
var greeter = new GreeterProxy(connection);
string greeting = await greeter.GreetAsync(Environment.UserName);

Console.WriteLine(greeting);

await connection.ShutdownAsync();
```

```csharp
// Server application

using IceRpc;
using IceRpc.Ice;
using IceRpc.Features;
using VisitorCenter;

await using var server = new Server(new Chatbot());
server.Listen();

// Wait until the console receives a Ctrl+C.
await CancelKeyPressed;
await server.ShutdownAsync();

// IGreeterService is an interface generated by the Ice compiler.
// The [IceService] attribute instructs the Ice Service source generator (provided by
// IceRpc.Ice.Generators.dll) to implement IDispatcher by directing "greet" requests to
// the GreetAsync method.
[IceService]
internal partial class Chatbot : IGreeterService
{
    public ValueTask<string> GreetAsync(
        string name,
        IFeatureCollection features,
        CancellationToken cancellationToken)
    {
        Console.WriteLine($"Dispatching greet request {{ name = '{name}' }}");
        return new($"Hello, {name}!");
    }
}
```
